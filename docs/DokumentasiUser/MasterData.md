# Master Data

Panduan penggunaan modul Master Data untuk mengelola data dasar sistem.

## Apa itu Master Data?
- Kumpulan data acuan yang dipakai oleh modul lain (contoh: dokter, poli, unit, pasien rujukan, obat, harga/tarif, perusahaan, dll).
- Perubahan di sini akan memengaruhi pilihan data di seluruh aplikasi.

## Akses Menu
- Buka `Master Data` dari Top Navbar atau menu utama.
- Gunakan kolom pencarian untuk menemukan item lebih cepat.

## Langkah Umum
- Tambah data: klik tombol `Tambah` atau `+`, isi form, lalu `Simpan`.
- Edit data: klik baris data atau tombol `Edit`, ubah isian, lalu `Simpan`.
- Hapus data: klik tombol `Hapus`, konfirmasi dengan benar.
- Filter/pencarian: ketik kata kunci pada kolom pencarian; gunakan filter jika tersedia.

## Contoh Pengelolaan
- Dokter: tambah data dokter (nama, SIP, spesialisasi, jadwal), pastikan nomor SIP valid.
- Poli/Unit: tambah poli beserta kode dan ketersediaan layanan.
- Obat: tambah data obat (nama, satuan, stok, harga, nomor batch jika perlu).
- Tarif/Harga: atur tarif tindakan sesuai kebijakan; perbarui saat ada perubahan biaya.

## Tips Penting
- Pastikan SIP dan masa berlaku diisi untuk tenaga kesehatan yang relevan.
- Gunakan pencarian sebelum menambah data baru untuk menghindari duplikasi.
- Lakukan penghapusan dengan hati-hati karena berdampak ke modul lain.

## Bantuan
- Hubungi admin jika menu yang diinginkan tidak tersedia atau Anda membutuhkan akses tambahan.
- Jika terjadi error, catat langkah yang dilakukan dan sampaikan ke admin untuk penelusuran.
