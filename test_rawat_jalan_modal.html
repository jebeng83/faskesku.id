Demo: RawatJalan Create Modal

Cara Penggunaan Modal

File Create.jsx telah berhasil dikonversi menjadi komponen modal popup dengan nama RawatJalanCreateModal.

Props yang Diperlukan:

  • isOpen (boolean): Mengontrol apakah modal ditampilkan atau tidak
  • onClose (function): Callback ketika modal ditutup
  • onSuccess (function): Callback ketika form berhasil disubmit
  • patients, polikliniks, dokters, dll.: Data yang diperlukan untuk form

Contoh Penggunaan:

import RawatJalanCreateModal from '@/Pages/RawatJalan/Create';

function ParentComponent() {
    const [showModal, setShowModal] = useState(false);
    
    const handleSuccess = () => {
        // Refresh data atau tampilkan notifikasi
        console.log('Data berhasil disimpan!');
    };
    
    return (
        <div>
            <button 
                onClick={() => setShowModal(true)}
                className="bg-blue-600 text-white px-4 py-2 rounded"
            >
                Tambah Registrasi
            </button>
            
            <RawatJalanCreateModal
                isOpen={showModal}
                onClose={() => setShowModal(false)}
                onSuccess={handleSuccess}
                patients={patients}
                polikliniks={polikliniks}
                dokters={dokters}
                // ... props lainnya
            />
        </div>
    );
}

Fitur Modal:

  • ✅ Ukuran full screen dengan scroll otomatis
  • ✅ Tombol close (X) di header
  • ✅ Menutup dengan tombol Escape
  • ✅ Backdrop click untuk menutup
  • ✅ Auto close setelah form berhasil disubmit
  • ✅ Responsive design
  • ✅ Dark mode support

Catatan: Modal ini menggunakan komponen Modal yang sudah ada di @/Components/Modal dan mengikuti pola yang sama dengan EmployeeCreateModal.
